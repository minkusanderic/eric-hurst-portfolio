Generate a PDF from HTML

Now

When constructing this site, I needed a way to export my Resume, that was formated in HTML, into a PDF that I could send to potential employers. I found many of the tools out there would use A4 as the default viewport size when formating the HTML to convert to PDF. While this worked, it did not give quite the feel that I was looking for. The viewport was too small, making the PDF a few pages long, and the text unneccesarily long. Enter wkhtmltopdf. This Linux tool takes in an HTML source and outputs a PDF. wkhtmltopdf uses the webkit engine under the hood, so it provides the flexability and compatablitiy that I needed. I was able to customize what the viewport that it used to render the HTML, giving me the look I needed. Here is how I used the command line: 

wkhtmltopdf --viewport-size 1000x1000 http://dev.regionsyx.com:8000/resume/ tmp.pdf

Beautiful is better than ugly.
